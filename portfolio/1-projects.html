Simon

A memory game based on an 80s classic in which the player has to repeat an ever-increasing series of visual/audio cues. Uses object-oriented JavaScript

Demo | Github

Tech: HTML, CSS, JavaScript, jQuery, Sound Manager2

CellarBook

Helping wine lovers keep track of their collection and write tasting notes about every bottle they drink.

This full-stack app uses four models, including "has many through" associations to manage each user's bottles. It employs a current_user method defined in the Application controller for user-related database queries and permissions.

Demo | Github

Tech: Ruby on Rails, Bcrypt, Foundation, Font Awesome, SASS, HTML, jQuery, CSS

TrackIt

For health enthusiasts to store their daily calorific intake, weight, exercise etc and view progress on a series of graphs.

This full-stack app uses Node, Express and EJS, with Chart.js for the graphs and a Passport local strategy for authentication.

Demo | Github

Tech: Node, Express, Mongoose, Bootstrap, Chart.js, HTML, CSS, JavaScript, jQuery

RoadTrip

A journey planner that displays routes and stopovers on maps and calculates distances and times for each step.

This MEAN-stack app uses angular-google-maps to access Google Maps API and avoid problems with asynchronicity. It uses a Passport local strategy for authentication.

Demo currently only available locally | Github

Tech: Node, Express, Mongoose, Angular, Bootstrap, Google Maps, Google Places, angular-google-maps, HTML, CSS, JavaScript, jQuery
